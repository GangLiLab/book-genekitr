5 Probe ID conversion

Microarray probes are used to measure the expression level of many genes. After getting microarray data, the first important thing before downstream analysis is probe id conversion.

genekitr::transProbe is built on Ensembl up-to-date annotation datasets and could join together with other genekitr functions elegantly.

Supported platforms can be listed by:

DT::datatable(hsapiens_probe_platform, options = list(pageLength = 10))

5.1 Probe id conversion

The user only needs to input probe id, organism, and wanted conversion type. Leave the rest to genekitr! It will keep the same order with input probes.

data(deg, package = "genekitr")
probes <- deg$probe_id

res <- transProbe(probes,"symbol",org = "human")
head(probes); head(res)
## [1] "8133876" "7965335" "7972259" "7972217" "8129573" "8015806"
##       probe_id symbol
## 6498   8133876   CD36
## 7208   7965335  DUSP6
## 1403   7972259    DCT
## 6687   7972217  SPRY2
## 1398   8129573  MOXD1
## 12679  8015806   ETV4

5.2 Convert to many types

User could define transTo as many types:

res2 <- transProbe(probes, c("symbol","ensembl"), org = "human")
head(res2)
##       probe_id         ensembl symbol
## 6498   8133876 ENSG00000135218   CD36
## 7208   7965335 ENSG00000139318  DUSP6
